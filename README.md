# Wind Speed Prediction Using Machine Learning Models

This project focuses on forecasting wind speed using Long Short-Term Memory (LSTM) networks and historical meteorological data to support renewable energy planning.

## Project Overview
The objective of this project is to develop a time-series prediction model that learns temporal patterns in wind data and provides accurate short-term forecasts.

## Methodology
- Collected historical weather data from public sources
- Cleaned and preprocessed time-series data
- Normalized features using Min-Max scaling
- Generated sequential input windows for LSTM
- Trained the model for multiple epochs
- Evaluated performance using regression metrics

## Model
- Algorithm: LSTM (Long Short-Term Memory)
- Framework: TensorFlow / Keras
- Optimizer: Adam
- Loss Function: Mean Squared Error (MSE)

## Results
The trained model demonstrated reliable performance in predicting wind speed under stable weather conditions, with reasonable accuracy on unseen data.
